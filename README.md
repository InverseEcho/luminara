# luminara
A custom AI I made for Perchance.

  Listing every Features in this code: 
  - Unrestricted Mode
  - Narrator Mode Toggle (integrated in chat)
  - Image Generation (using Character Description)
  - Regenerate Response
  - Streaming Text Response
  - Dark Mode (Improved UI)
  - Custom Character Creation (Name, Gender, Description, System Prompt, First Message, Pre-instruction, Reminder)
  - User Profile Editing
  - Sidebar Contact List
  - Main Menu (Settings, Profile, Import/Export) accessed via sidebar button
  - Chat bubble actions (Continue, Edit, Regenerate, Custom Regenerate, Delete)
  - AI Settings (Tone, Length, Creativity, Unrestricted Mode)
  - Quick Actions (Character Initiate, User Simulate, Narrator)
  - Responsive design for desktop and mobile
  - Create Character from URL (Auto-fill data) + Optional Instructions
  - Image Generation from Chat Context (Auto-summarize)
  - Character List Window with Card View and Actions (Start Chat, Export, Edit, Delete)
  - Import Character JSON file in character creation
  - Image Overflow Fix (Max-height constraints + Aspect Ratio preservation)
  - Dual Image Generation (1:1 and 16:9) in a single Carousel Bubble
  - Toggle Prompt Button for Image Bubbles
  - Quick AI Settings Access (Slider Panel)
  - Enhanced Image Generation UI
  - AI Context Window (System + User + Chat Context)
  - Smart Message Summarization for Image Prompts
  - Character Creation Quick Edit Button (Inline)
  - Character List Sorting & Filtering (Name, Created Date)
  - Settings Persistence (LocalStorage)
  - UI & Performance Improvements
  - 60-30-10 Color Rule (60% neutral BG, 30% surfaces, 10% action)
  - WCAG AA Accessibility (4.5:1 contrast ratio on chat bubbles)
  - TIME SKIP / REWIND
  - CHARACTER AVATAR
